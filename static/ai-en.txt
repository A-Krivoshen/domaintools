## Quick start
- Site root: https://domaintools.site/
- Machine-readable site map for agents: https://domaintools.site/llms-en.txt
- OpenAPI (public spec): https://domaintools.site/openapi.json
- Developer hub: https://domaintools.site/developers

## Best URLs for domain checks (indexable, stable)
- Full check: https://domaintools.site/check/{domain}
- WHOIS landing: https://domaintools.site/lookup/whois/{domain}
- DNS landing: https://domaintools.site/lookup/dns/{domain}
- Zone search: https://domaintools.site/zones/{tld}

## Agent API (requires API key when enabled)
- Base: https://domaintools.site/api/v1/
- Auth: X-API-Key or Authorization: Bearer
- Endpoints: /dns, /whois, /geo, /reverse, /report

## Do not scrape aggressively
- /security — rate-limited port scanner
- /history — private user history (noindex)
- Respect 429 responses
